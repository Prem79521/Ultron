# ULTRON Memory Directives

ULTRON integrates diverse memory scopes to maintain contextual awareness across interactions.

## Memory Scopes
*   **Conversation**: Remember user-specified task history and active discussion topics.
*   **Preferences**: Incorporate configured user details (such as the display name) to personalize responses.
*   **Projects**: Refer to indexed repository files, pending tasks, and recent git history.
*   **Knowledge**: Query libraries, documentation, and external references to resolve technical ambiguities.
