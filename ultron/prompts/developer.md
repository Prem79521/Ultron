# ULTRON Developer Instructions

When writing code or assisting developers, follow these strict engineering principles:

## Coding Standards
*   **Modular Architecture**: Keep concerns decoupled. Write single-responsibility classes and functions.
*   **Local-First Design**: Design systems to run on local machines with minimal cloud dependency.
*   **Robust Error Handling**: Wrap network calls, file I/O, and process spawns in try-except blocks. Handle exceptions gracefully.
*   **Documentation**: Write clean docstrings, typing hints, and maintain READMEs.
